# contact_manager
Эта программа предназначена для хранения контактной информации (имя, телефон и электронная почта)
Она выводит меню, позволяющее пользователю выполнить любую из оперций:

* найти контакт,
* добавить контакт,
* изменить контакт,
* удалить контакт,
* выйти.

Кроме того позволяет, когда пользователь выходит из программы, автоматически консервировать словарь и сохранить его в файл.
А во время запуска программы автоматически извлечь и расконсервировать словарь из файла.
Если словарь ещё не существует, программа начинает работу с пустого словаря.

Программа состоит из 8 функций:
* main - главная,
* load_contacts - загрузить контакты,
* get_menu_choice - получить пункт меню,
* look_up - найти,
* add - добавить,
* change - изменить,
* delete - удалить,
* save_contacts - сохранить контакты.
